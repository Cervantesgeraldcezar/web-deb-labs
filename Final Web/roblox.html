Introduction

This website presents the research project titled "Roblox as an Instructional Material in Teaching ICT". The study explores how game-based learning—through a custom Roblox Obby—can enhance student understanding of HTML concepts.

Key Features of the Roblox Instructional Game

  • Interactive Roblox Obby designed around HTML lessons
  • Obstacle-based progression linked to ICT competencies
  • Embedded quizzes and automatic feedback
  • Gamification elements: badges, points, and rewards
  • Student progress tracking and performance monitoring

Development Process

The system was developed following four major stages:

  1. Design Concept: Planning game flow, ICT learning objectives, and lesson integration.
  2. Development: Building the Roblox world, scripting interactions, and embedding HTML lessons.
  3. Testing: Conducting functionality tests, user testing, and revisions.
  4. Evaluation: Assessing curriculum alignment, usability, accessibility, and learning effectiveness.

Evaluation Framework

The system is evaluated based on:

  • Curriculum Alignment with CHED ICT Outcomes
  • Accessibility and user experience
  • Functionality and stability
  • Learning effectiveness and motivation

Rating uses Likert scale and includes statistical analysis such as mean and standard deviation.
